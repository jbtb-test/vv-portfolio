# V&V AI Portfolio

Ce dépôt constitue le **portfolio principal** du projet V&V IA.

## Objectif
Démontrer une maîtrise complète du **V-cycle** (exigences, tests, couverture, KPI, amélioration continue),
avec **IA utilisée comme accélérateur contrôlé** :
- IA désactivée par défaut
- IA = moteur de suggestion uniquement
- Décision humaine systématique

## Contenu
- PLAN.ods : plan maître du projet (baseline)
- docs/ : architecture, V-model IA, décisions et risques
- Liens vers les 3 applications V&V :
  - QRA (Quality Risk Assessment)
  - TCTC (Test Case & Test Coverage)
  - AITA (AI Test Assistant)

## Méthode
- Exécution ligne par ligne
- DoD explicite par étape
- Validation manuelle
- Traçabilité décisions & risques

Ce dépôt est le point dentrée **recruteur**.
